# 4.8 저장JSON 구현하기

> **IBSheet7 개발자 가이드** > 개발 절차 > 4.8 저장JSON 구현하기

---

> 저장 함수를 호출하면 인자로 넘긴 URL로 페이지에 연결한다.
>
> 저장은 JSON 형식을 사용하지 않고 기존 param형식을 사용한다.
>
> 저장 처리 후 처리 결과는 JSON 형태로 구성되며 페이지의 내용을 읽어 들여 결과를 표현한다.

- **저장 처리 결과 표시 형태**

> 저장 처리를 완료하고 전체 데이터에 대해서 처리 완료 여부를 JSON으로 반환하는 형태이다

```
{
Result : {Code:0, Message:'저장되었습니다.'}
}
```

- **처리 결과 설정**

> 저장 처리 결과로 설정할 수 있는 값은 다음과 같다.

| 트랜잭션 결과 | 처리 결과   |           |                    |
| ------- | ------- | --------- | ------------------ |
| 트랜잭션 결과 | 트랜잭션 상태 | 메세지 표시 결과 |                    |
| Code    | 양수:성공   | Message   | "저장되었습니다" 표시       |
| Code    | 음수:실패   | Message   | "저장이 실패 되었습니다." 표시 |

**조회/저장 자료규조에 대한 자세한 내용을 개발자가이드 6장을 참고.  
조회 함수 (DoSearch,LoadSearchData)호출 후에는 OnSearchEnd 이벤트가 발생한다.  
저장 함수 (DoSave,LoadSaveData) 호출 후에는 OnSaveEnd 이벤트가 발생한다.**
