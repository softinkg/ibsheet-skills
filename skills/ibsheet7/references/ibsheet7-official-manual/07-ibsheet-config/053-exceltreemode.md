# ExcelTreeMode

> **IBSheet7 개발자 가이드** > IBSheet Config > ExcelTreeMode

---

엑셀 다운시 트리의 접혀진 행을 다운 받을지 설정

| 설정값 | 내용                                            |
| --- | --------------------------------------------- |
| 0   | 트리의 접혀진 행을 다운 안함 (Default)                    |
| 1   | 트리의 접혀진 행도 모두 다운 SetRowHidden 으로 숨긴 로우도 모두 다운 |
