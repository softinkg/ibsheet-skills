# InvalidInputBehavior

> **IBSheet7 개발자 가이드** > IBSheet Config > InvalidInputBehavior

---

편집 완료시점에 입력값이 invalid 인 경우에 대한 처리 방법 설정

| 설정값 | 내용                                   |
| --- | ------------------------------------ |
| 0   | alert 메시지 후 해당 셀 편집상태 유지 (Default)   |
| 1   | alert 메시지 후 해당 셀 편집종료(입력값 취소 처리)     |
| 2   | alert 메시지 후 해당 셀 편집상태 유지 (입력값 취소 처리) |
