# CalButtons

> **IBSheet7 개발자 가이드** > IBSheet Config > CalButtons

---

달력팝업의 사용할 버튼 설정

사용하고자 하는 버튼을 "|" 로 연결하여 설정한다.

이 설정은 시트내의 달력팝업에만 적용된다.

| 설정값        | 내용         |
| ---------- | ---------- |
| Close      | 취소 버튼      |
| Today      | 오늘일자 입력 버튼 |
| Yesterday  | 어제일자 입력 버튼 |
| InputEmpty | 빈값 입력 버튼   |
| Ok         | 선택 버튼      |
