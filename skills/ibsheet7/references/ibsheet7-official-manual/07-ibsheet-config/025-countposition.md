# CountPosition

> **IBSheet7 개발자 가이드** > IBSheet Config > CountPosition

---

건수정보 표시 여부 및 위치 설정

| 설정값 | 내용                 |
| --- | ------------------ |
| 0   | 표시 하지 않음 (Default) |
| 1   | 좌측 상단              |
| 2   | 우측 상단              |
| 3   | 좌측 하단              |
| 4   | 우측 하단              |
