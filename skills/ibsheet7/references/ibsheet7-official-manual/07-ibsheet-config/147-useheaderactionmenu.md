# UseHeaderActionMenu

> **IBSheet7 개발자 가이드** > IBSheet Config > UseHeaderActionMenu

---

헤더 컨텍스트 메뉴 사용여부 설정

헤더메뉴를 사용함으로 설정하고 헤더메뉴를 설정하지 않는경우 자동으로 기본 메뉴가 출력된다.

(HeaderActionMenu 참조)

| 설정값 | 내용                 |
| --- | ------------------ |
| 0   | 속성 사용 안함 (Default) |
| 1   | 속성 사용함             |
