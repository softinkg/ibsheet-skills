# ### ClearFilterRow Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > ### ClearFilterRow Method

---

- **기능**

> 필터 행을 초기화한다.
>
> 초기화시 필터링이 되어있는 경우 해당 필터링은 모두 초기화가 된다.

- **Syntax**

|        |                            |
|:------:|----------------------------|
| Syntax | ObjId.**ClearFilterRow**() |

- **Info**

> **Return**: (없음)

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
// 필터행 초기화
mySheet.ClearFilterRow();
```

- **제공 버전**

7.0.13.96
