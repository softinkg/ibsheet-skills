# SetFilterValue Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > SetFilterValue Method

---

- **기능**

> 필터행 사용시 컬럼의 필터값을 설정할 때 사용한다.

- **Syntax**

|        |                                                         |
|:------:|---------------------------------------------------------|
| Syntax | ObjId.**SetFilterValue**(Col, Value, Option, [Event]) |

- **Info**

> **Return**: None

| Parameter | Type            | 필수여부 | 설 명                                |     |
| --------- | --------------- | ---- | ---------------------------------- | --- |
| Col       | Number / String | 필수   | 컬럼의 Index 또는 SaveName              |     |
| Value     | String          | 필수   | 필터에 설정할 값                          |     |
| Option    | Number          | 선택   | 설정 할 Option 값                      |     |
| Event     | Boolean         | 선택   | OnFilterEnd이벤트 발생 여부 (Default : 1) |     |

- **Option Value**

| **0**  | **사용 안 함**  |        |                 |
| ------ | ----------- | ------ | --------------- |
| **1**  | **같음**      | **2**  | **같지 않음**       |
| **3**  | **작음**      | **4**  | **같거나 작음**      |
| **5**  | **큼**       | **6**  | **같거나 큼**       |
| **7**  | **단어로 시작함** | **8**  | **단어로 시작하지 않음** |
| **9**  | **단어로 끝남**  | **10** | **단어로 끝나지 않음**  |
| **11** | **포함함**     | **12** | **포함하지 않음**     |

- **Example**

```
// 5컬럼에 "서울"이 포함된 문자열 필터링하기
mySheet.SetFilterValue(5, "서울", 11);
```
