# getSheetWidth ***(method)***

> 시트의 너비를 반환합니다. 


### Syntax
```javascript
void getSheetWidth();
```

### Return Value
***number*** 시트의 높이

### Example
```javascript
// 시트의 높이를 반환합니다.
sheet.getSheetWidth();
```
