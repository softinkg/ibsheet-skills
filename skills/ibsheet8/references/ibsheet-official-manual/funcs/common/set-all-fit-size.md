> fitSize 를 이용하여 전체 컬럼의 너비를 조정합니다. 


### Syntax
```javascript
void setAllFitSize();
```

### Return Value
***none***

### Example
```javascript
// fitSize를 이용하여 전체 컬럼의 너비를 조정합니다.
sheet.setAllFitSize();
```
