# getLocale ***(method)***
> 시트에 설정된 언어 코드를 가져오는 메소드입니다. 


### Syntax
```javascript
void getLocale();
```

### Return Value
***string*** : 시트에 설정된 언어 코드

### Example
```javascript
sheet.getLocale();
```
