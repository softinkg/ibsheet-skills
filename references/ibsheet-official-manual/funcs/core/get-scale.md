# getScale ***(method)***
> 시트의 배율(zoom factor)정보를 확인합니다.

### Syntax
```javascript
number getScale();
```


### Return Value
***number*** : 시트에 설정된 배율 정보

### Example
```javascript
sheet.getScale();
```
