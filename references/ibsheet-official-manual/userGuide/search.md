# 찾기 기능

그리드 내에서 찾기 다이얼로그를 띄워 원하는 데이터를 검색하거나, 원하는 내용으로 바꿀 수 있습니다. 

해당 기능은 데이터 내에서 텍스트 기준으로 검색/변경이 가능합니다. 


## *검색 / 바꾸기*

**검색 기능** 


시트에서 ctrl+F 키를 눌러, 찾기 다이얼로그를 사용하실 수 있습니다.

![찾기다이얼로그](/assets/imgs/showFindDialog_1.png)
