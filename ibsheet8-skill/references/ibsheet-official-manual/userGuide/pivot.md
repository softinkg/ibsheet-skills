# 피벗 기능

피벗 다이얼로그를 통해 사용자가 피벗팅 할 데이터를 구성하고, 계산 결과를 확인하실 수 있습니다.


## *피벗 다이얼로그*

대상에 맞게 컬럼 기준을 설정한 뒤 피벗 테이블을 생성할 수 있습니다.

시트에서 ctrl+alt+t 키를 눌러, 피벗 다이얼로그를 사용하실 수 있습니다.
