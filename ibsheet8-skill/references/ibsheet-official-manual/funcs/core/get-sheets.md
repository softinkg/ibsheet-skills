# !getSheets ***(method)***

> 스프레드 시트에 있는 탭들의 목록을 가져옵니다.


### Syntax
```javascript
object getSheets( );
```

### Return Value
***object*** : 시트에 있는 탭 목록

### Example
```javascript
// 시트에 있는 탭 목록 확인
sheet.getSheets();
```
