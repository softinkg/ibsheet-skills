# getSheetFocused ***(method)***

> 현재 포커스된 시트 객체를 반환합니다. 


### Syntax
```javascript
void getSheetFocused();
```

### Return Value
***Sheet*** 시트 객체


### Example
```javascript
// 푸터 영역에 있는 푸터 데이터 로우 객체들의 배열을 반환합니다.
sheet.getSheetFocused();
```
