# HideFilterRow Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > HideFilterRow Method

---

- **기능**

> IBSheet 상단 고정행의 필터행을 삭제한다.
>
> 필터행이 있는 경우 삭제되고, 없으면 아무일도 발생하지 않는다.

- **Syntax**

|        |                           |
|:------:|---------------------------|
| Syntax | ObjId.**HideFilterRow**() |

- **Info**

> **Return**: None

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
//필터행 삭제하기
mySheet.HideFilterRow();
```

- **제공 버전**

7.0.0.0
