# GetDataRows Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > GetDataRows Method

---

- **기능**

> 단위 데이터 행의 설정 개수를 확인 한다.

- **Syntax**

|        |                         |
|:------:|-------------------------|
| Syntax | ObjId.**GetDataRows**() |

- **Info**

> **Return**: Integer, 설정 되어 있는 단위 데이터 행의 개수

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```javascript
// 단위데이터행의 개수를 확인한다.
var dataRows = mySheet.GetDataRows();.
```

- **제공 버전**
