# ClearSelection Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > ClearSelection Method

---

- **기능**

> 마우스 드래그를 이용한 선택 영역을 초기화 한다.

- **Syntax**

|        |                            |
|:------:|----------------------------|
| Syntax | ObjId.**ClearSelection**() |

- **Info**

> **Return**: None

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
// 선택 영역을 초기화 한다.
mySheet.ClearSelection()
```

- **제공 버전**

7.0.0.0
