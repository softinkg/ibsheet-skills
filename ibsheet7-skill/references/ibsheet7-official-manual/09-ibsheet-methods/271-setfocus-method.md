|        |                      |
|:------:|----------------------|

> **Return**: None

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
// 시트에 포커스를 설정한다.
mySheet.SetFocus();
```
