# DoPrint Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > DoPrint Method

---

- **기능**

> 보여지고 있는 모든 데이터를 인쇄한다.
>
> 인쇄 처리는 브라우저의 인쇄 기능을 따르며 배경색 및 이미지를 인쇄 하기 위해서는 브라우저의 인쇄설정을 변경 하여야 한다.

- **Syntax**

|        |                     |
|:------:|---------------------|
| Syntax | ObjId.**DoPrint**() |

- **Info**

> **Return**: None

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
// 인쇄 하기
mySheet.DoPrint();
```

- **제공 버전**

> **7.0.0.0**
