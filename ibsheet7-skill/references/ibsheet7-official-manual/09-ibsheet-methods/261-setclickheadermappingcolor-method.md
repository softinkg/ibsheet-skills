# SetClickHeaderMappingColor Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > SetClickHeaderMappingColor Method

---

- **기능**

> 셀 클릭시 셀에 매핑되는 헤더셀의 색상을 설정한다.

- **Syntax**

|        |                                              |
|:------:|----------------------------------------------|
| Syntax | ObjId.**SetClickHeaderMappingColor**(color); |

- **Info**

> **Return**: None

| Parameter | Type   | 필수여부 | 설 명    |
| --------- | ------ | ---- | ------ |
| color     | String | 필수   | 사용할 색상 |

- **Example**

```
// 셀 클릭시 셀에 매핑되는 헤더셀의 색상을 설정
mySheet.SetClickHeaderMappingColor(‘#ffffff’);
```
