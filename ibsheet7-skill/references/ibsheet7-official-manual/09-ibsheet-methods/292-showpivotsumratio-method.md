# ShowPivotSumRatio Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > ShowPivotSumRatio Method

---

- **기능**

> 피벗 테이블에 대한 합계 비율을 적용 한다.
>
> 이 기능은 피벗 테이블 결과가 적용된 시트에서만 사용이 가능 하다.

- **Syntax**

|        |                                   |
|:------:|-----------------------------------|
| Syntax | ObjId.**ShowPivotSumRatio**(Mode) |

- **Info**

> **Return**: None

| Parameter | Type   | 필수여부 | 설 명                                                                                                                                                        |     |
| --------- | ------ | ---- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| Mode      | Number | 필수   | 합계 비율 모드 (Default : 0) \| 설정 값 \| 설명      \| \| ---- \| ------- \| \| 0    \| 기본 합계   \| \| 1    \| 총 합계 비율 \| \| 2    \| 열 합계 비율 \| \| 3    \| 행 합계 비율 \| |     |

- **Example**

```javascript
// 피벗 테이블 구성 완료시점에 총합계 비율을 설정 한다.
Function mySheet_Pivot_OnSearchEnd() {
mySheet_Pivot.ShowPivotSumRatio(1);
}
```
