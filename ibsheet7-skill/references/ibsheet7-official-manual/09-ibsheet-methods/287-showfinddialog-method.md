# ShowFindDialog Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > ShowFindDialog Method

---

- **기능**

> 사용자가 시트에서 ctrl+shift+F 입력시 표시되는 찾기 창을 표시한다.

- **Syntax**

|        |                            |
|:------:|----------------------------|
| Syntax | ObjId.**ShowFindDialog**() |

- **Info**

> **Return**: None

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
//찾기창을 표시한다.
mySheet.ShowFindDialog();
```

- **제공 버전**

7.0.0.0
