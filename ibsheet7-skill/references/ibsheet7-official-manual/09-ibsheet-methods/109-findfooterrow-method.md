# FindFooterRow Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > FindFooterRow Method

---

- **기능**

> 푸터행의 인덱스를 반환한다.

- **Syntax**

|        |                           |
|:------:|---------------------------|
| Syntax | ObjId.**FindFooterRow**() |

- **Info**

> **Return**: Integer, 푸터행의 인덱스 (Default=-1)

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```javascript
//푸터행의 행 번호를 가져온다.
var footerIdx = mySheet.FindFooterRow();
```
