# IsFocused Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > IsFocused Method

---

- **기능**

> 시트의 포커스 상태를 확인 한다.

- **Syntax**

|        |                       |
|:------:|-----------------------|
| Syntax | ObjId.**IsFocused**() |

- **Info**

> **Return**: Boolean, 포커스 상태.

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
// 시트의 Focus 여부 확인
console.log("focused", mySheet.IsFocused());
```
