# GoToBaseSheet Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > GoToBaseSheet Method

---

- **기능**

> 피벗 설정 팝업으로부터 피벗 테이블이 화면에 표시되어 있는 경우 이 기능을 사용하면 해당 피벗 테이블의 원본 시트로 이동 한다.
>
> 피벗 테이블 설정 팝업의 [원본시트이동] 버튼의 기능과 동일하게 동작 한다.

- **Syntax**

|        |                           |
|:------:|---------------------------|
| Syntax | ObjId.**GoToBaseSheet**() |

- **Info**

> **Return**: None

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
// 피벗 테이블의 원본 시트로 이동
mySheet.GoToBaseSheet();
```

- **제공 버전**

> **7.0.0.0**
