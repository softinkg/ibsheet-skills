# ReturnColumnPos Method

> **IBSheet7 개발자 가이드** > IBSheet Methods > ReturnColumnPos Method

---

- **기능**

> 이동된 컬럼의 위치를 처음 위치로 되돌린다.

- **Syntax**

|        |                             |
|:------:|-----------------------------|
| Syntax | ObjId.**ReturnColumnPos**() |

- **Info**

> **Return**: None

| Parameter | Type | 필수여부 | 설 명 |
| --------- | ---- | ---- | --- |
| 없음        |      |      |     |

- **Example**

```
//초기 위치로 되돌린다.
mySheet.ReturnColumnPos();
```

- **제공 버전**

> **7.0.0.0**
