# OnLoadExcel Event

> **IBSheet7 개발자 가이드** > IBSheet Events > OnLoadExcel Event

---

- **기능**

> LoadExcel 처리가 완료된 후에 이벤트가 발생한다.
>
> Code, Msg 인자는 실패의 경우에만 해당 값이 전달 된다.

- **Syntax**

|        |                                                            |
|:------:|------------------------------------------------------------|
| Syntax | **function 오브젝트ID_OnLoadExcel**(result, code, msg) { } |

- **Parameters**

| Parameter | Type    | 설 명                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| --------- | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| result    | Boolean | 로드한 결과. 성공 : true, 실패 : false                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Code      | Number  | 결과 코드 \| code \| Description                                                                   \| \| ---- \| ----------------------------------------------------------------------------- \| \| -1   \| LoadExcel.jsp 에서 setMaxRows로 설정한 갯수보다 더 많은 행을 로드 하는 경우                        \| \| -2   \| LoadExcel.jsp 에서 setStrictHeaderMatch설정 후, 시트 헤더가 엑셀에 하나라도 존재하지 않는 경우         \| \| -3   \| LoadExcel.jsp 에서 setMaxColumns 로 설정한 갯수보다 더 많은 열을 로드 하는 경우                    \| \| -10  \| LoadExcel.jsp 에서 setMaxFileSize 로 설정한 사이즈보다 더 큰 파일을 로드 하려는 경우(2.X.X.jar 만 지원) \| \| -500 \| 엑셀 로드시 오류가 발생한 경우                                                             \| \| 그외   \| 사용자 코드                                                                        \| |
| Msg       | String  | 결과 메시지 Ibmsg 파일의 값으로 메시지 전달 \| code \| Description                   \| \| ---- \| ----------------------------- \| \| -1   \| Ibmsg 파일의 SYS_OverMaxRow      \| \| -2   \| Ibmsg 파일의 SYS_OverMaxColumns  \| \| -3   \| Ibmsg 파일의 SYS_NoMatchedHeader \| \| -10  \| Ibmsg 파일의 SYS_OverMaxFileSize \| \| -500 \| Ibmsg 파일의 SYS_LoadExcelError  \| \| 그외   \| 사용자 메시지                       \|                                                                                                                                                                                                                                                                                                                                                                           |

- **Example**

```javascript
function mySheet_OnLoadExcel(result,code,msg) {
if(result) {
alert('엑셀 로딩이 완료되었습니다.');
} else {
alert(msg);
}
}
```
