# UseHeaderSortCancel

> **IBSheet7 개발자 가이드** > IBSheet Config > UseHeaderSortCancel

---

헤더 클릭을 통한 컬럼 Sort 처리시 취소 모드 사용 여부

| 설정값 | 내용                                          |
| --- | ------------------------------------------- |
| 0   | 속성 사용 안함 (Default) 헤더 클릭시 Asc / Desc 순으로 처리 |
| 1   | 속성 사용함 헤더 클릭시 Asc / Desc / 취소 순으로 처리        |
