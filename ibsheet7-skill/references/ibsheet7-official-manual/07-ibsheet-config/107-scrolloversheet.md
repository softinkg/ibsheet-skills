# ScrollOverSheet

> **IBSheet7 개발자 가이드** > IBSheet Config > ScrollOverSheet

---

시트 내의 마우스 휠을 이용한 스크롤 이벤트를 부모 element로 전달 여부.

| 설정값 | 내용                                                        |
| --- | --------------------------------------------------------- |
| 0   | 시트에서만 스크롤 이동(Default)                                     |
| 1   | 시트 내에서 마우스 휠로 스크롤 이벤트가 발생할 때 양 끝에서 부모 element로 스크롤 이벤트 전달 |
