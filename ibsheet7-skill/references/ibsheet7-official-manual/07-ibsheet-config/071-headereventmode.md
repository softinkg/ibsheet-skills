# HeaderEventMode

> **IBSheet7 개발자 가이드** > IBSheet Config > HeaderEventMode

---

헤더의 이벤트 발생종류 설정

| 설정값 | 내용                             |
| --- | ------------------------------ |
| 0   | 모든 마우스 이벤트 발생 안함               |
| 1   | 모든 마우스 이벤트 발생함 (Default)       |
| 2   | MouseDown, MouseMove 이벤트만 발생 함 |
