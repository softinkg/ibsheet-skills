# TouchScroll

> **IBSheet7 개발자 가이드** > IBSheet Config > TouchScroll

---

모바일에서 스크롤(가로/세로) 디자인 설정

| 설정값 | 내용                               |
| --- | -------------------------------- |
| 0   | 스크롤바 영역만 있을 뿐 실제 스크롤바를 통한 이동은 불가 |
| 1   | 18px 크기의 이미지 스크롤바 디자인            |
| 2   | 10px 크기의 이미지 스크롤바 디자인 (Default)  |
| 3   | 9px 크기의 이미지 스크롤바 디자인             |
