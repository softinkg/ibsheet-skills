# HeaderSortActionMode

> **IBSheet7 개발자 가이드** > IBSheet Config > HeaderSortActionMode

---

단일 컬럼 정렬, 다중 컬럼 정렬을 위한 헤더의 사용자 입력 방식 모드 설정

| 설정값         | 입력 방식        | 내용       |
| ----------- | ------------ | -------- |
| 0 (Default) | Click        | 다중 컬럼 정렬 |
| 0 (Default) | Ctrl + Click | 단일 컬럼 정렬 |
| 1           | Click        | 단일 컬럼 정렬 |
| 1           | Ctrl + Click | 다중 컬럼 정렬 |
