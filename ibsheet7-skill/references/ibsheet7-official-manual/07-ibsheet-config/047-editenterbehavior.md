# EditEnterBehavior

> **IBSheet7 개발자 가이드** > IBSheet Config > EditEnterBehavior

---

편집중 엔터키 입력에 대한 동작 설정

| 설정값     | 내용                              |
| ------- | ------------------------------- |
| tab     | 편집 완료 후 오른쪽 셀로 포커스 이동 (Default) |
| down    | 편집 완료 후 아래 셀로 포커스 이동            |
| newline | 줄바꿈되어 개행 처리 됨                   |
| none    | 편집 완료 후 포커스 유지                  |
| editTab | 편집 완료 후 다음 편집 가능한 셀로 이동         |
