# DragMode

> **IBSheet7 개발자 가이드** > IBSheet Config > DragMode

---

드래그 처리 방법 설정

| 설정값 | 내용                                             |
| --- | ---------------------------------------------- |
| -1  | 드래그 사용 안함                                      |
| 0   | 일반 : 셀 또는 행 범위 셀렉션 Ctrl 키 이용 : 행 드래깅 (Default) |
| 1   | 일반 : 행 드래깅 Ctrl 키 이용 : 셀 또는 행 범위 셀렉션           |
