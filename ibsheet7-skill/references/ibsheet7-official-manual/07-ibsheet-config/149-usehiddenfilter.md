# UseHiddenFilter

> **IBSheet7 개발자 가이드** > IBSheet Config > UseHiddenFilter

---

필터 행 없이 필터링 사용 여부 설정

(SetFilterValue 참조)

| 설정값 | 내용                                                |
| --- | ------------------------------------------------- |
| 0   | 속성 사용 안함 (Default)                                |
| 1   | 속성 사용함 필터행이 보이지 않아도 SetFilterValue를 사용 할 수 있게 설정. |
