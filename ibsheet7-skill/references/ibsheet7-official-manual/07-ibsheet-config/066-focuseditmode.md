# FocusEditMode

> **IBSheet7 개발자 가이드** > IBSheet Config > FocusEditMode

---

포커스시 편집여부 설정

| 설정값 | 내용                                  |
| --- | ----------------------------------- |
| 0   | (기본값)                               |
| 1   | 포커스시 편집 모드                          |
| 2   | 포커스시 편집 모드 (Combo, ComboEdit 타입 제외) |
