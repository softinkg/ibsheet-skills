# SearchMode

> **IBSheet7 개발자 가이드** > IBSheet Config > SearchMode

---

조회 모드 설정

| 설정값 | 내용                                          |
| --- | ------------------------------------------- |
| 0   | smGeneral – 일반 조회 모드                        |
| 1   | smClientPaging – 페이징 모드                     |
| 2   | smLazyLoad – Lazy Load 모드 (Default)         |
| 3   | smServerPaging – 실시간 서버 처리 모드 (스크롤링 방식)     |
| 4   | smServerPaging2 – 실시간 서버 처리 모드 (페이지 인덱스 방식) |
