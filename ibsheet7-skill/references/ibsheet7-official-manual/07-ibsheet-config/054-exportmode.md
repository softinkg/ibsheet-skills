# ExportMode

> **IBSheet7 개발자 가이드** > IBSheet Config > ExportMode

---

엑셀 Import/Export에 사용할 모듈을 설정

클라이언트 모듈의 경우 ibexcel.js 파일 필요.

| 설정값 | 내용                                      |
| --- | --------------------------------------- |
| 1   | 서버모듈 (Default)                          |
| 2   | 클라이언트 IE10 이상 지원                        |
| 3   | 서버모듈, 클라이언트모듈 (ibexcel.js가 없으면 서버모듈 동작) |
