# ClipPasteMode

> **IBSheet7 개발자 가이드** > IBSheet Config > ClipPasteMode

---

클립보드 붙여넣기 방법 설정

| 설정값 | 내용                              |
| --- | ------------------------------- |
| -1  | 사용안함                            |
| 0   | 선택 셀 (Default)                  |
| 1   | 선택 셀 기준 구분자 처리                  |
| 2   | 선택 셀 기준 구분자 (부족한 경우 행 추가)       |
| 3   | 신규 행                            |
| 4   | 선택 영역의 크기와 관계없이 클립보드 내용 붙여넣기 처리 |
*※ 설정값 세부설명은 ClipPasteMode() 메서드 참조*
