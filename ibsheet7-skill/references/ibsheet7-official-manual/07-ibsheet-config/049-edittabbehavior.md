# EditTabBehavior

> **IBSheet7 개발자 가이드** > IBSheet Config > EditTabBehavior

---

편집중 탭키 입력에 대한 동작 설정

| 설정값 | 내용                                               |
| --- | ------------------------------------------------ |
| 0   | 편집 가능한 다음 셀로 이동 (Default)                        |
| 1   | 편집여부와 상관없이 다음 셀로 이동 (다음셀이 편집 가능한 경우면 편집상태 유지)    |
| 2   | 편집여부와 상관없이 다음 셀로 이동 (다음셀의 편집 가능여부와 무관하게 편집상태 종료) |
