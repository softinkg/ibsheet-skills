# DownWaitImageVisible

> **IBSheet7 개발자 가이드** > IBSheet Config > DownWaitImageVisible

---

파일 다운 중 대기 이미지 출력여부 설정

출력 안함으로 설정 시 OnDownFinish 이벤트도 발생하지 않는다.

| 설정값 | 내용                        |
| --- | ------------------------- |
| 0   | 대기이미지, 이벤트 출력 안함          |
| 1   | 대기이미지, 이벤트 출력 함 (Default) |
