# CustomScroll

> **IBSheet7 개발자 가이드** > IBSheet Config > CustomScroll

---

가로/세로 스크롤바에 대한 디자인 설정.

| 설정값 | 내용                             |
| --- | ------------------------------ |
| 0   | 18px 크기의 div 스크롤바 디자인(Default) |
| 1   | 18px 크기의 이미지 스크롤바 디자인          |
| 2   | 10px 크기의 이미지 스크롤바 디자인          |
| 3   | 9px 크기의 이미지 스크롤바 디자인           |
