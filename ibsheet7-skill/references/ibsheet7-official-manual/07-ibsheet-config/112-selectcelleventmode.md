# SelectCellEventMode

> **IBSheet7 개발자 가이드** > IBSheet Config > SelectCellEventMode

---

조회된 데이터가 없는 경우에도 OnSelectCell 이벤트를 발생시킬지 여부를 설정

| 설정값 | 내용                               |
| --- | -------------------------------- |
| 0   | OnSelectCell 이벤트 발생 안함 (Default) |
| 1   | OnSelectCell 이벤트 발생              |
