# EditableColorDiff

> **IBSheet7 개발자 가이드** > IBSheet Config > EditableColorDiff

---

편집이 불가능한 셀을 색상으로 구분하여 표시할지 여부를 설정

| 설정값 | 내용                                 |
| --- | ---------------------------------- |
| 0   | 편집불가능한 셀을 구분없이 표시                  |
| 1   | 편집불가능한 셀을 css에서 설정한 색상으로 표시        |
| 2   | 편집불가능한 셀을 css의 설정값과 기본배경색의 조합으로 표시 |
