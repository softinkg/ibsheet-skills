# HeaderSort

> **IBSheet7 개발자 가이드** > IBSheet Config > HeaderSort

---

헤더 클릭을 통한 컬럼 Sort 처리 방법 설정

| 설정값 | 내용                          |
| --- | --------------------------- |
| 0   | 사용 안함                       |
| 1   | Sort 기능 사용(Default)         |
| 2   | Sort 아이콘만 표시                |
| 3   | colSpan 아닌 헤더 셀만 Sort 기능 사용 |
